## Smart Gym Injury Prevention System

This project uses AI and computer vision to detect incorrect gym posture and help prevent injuries.

## Features

* Real-time pose detection
* Squat posture analysis
* Live feedback (Correct / Incorrect)

## Technologies Used

* Python
* OpenCV
* MediaPipe
* Streamlit

## How to Run

1. Install requirements:
   pip install opencv-python mediapipe numpy streamlit

2. Run the app:
   streamlit run app.py
